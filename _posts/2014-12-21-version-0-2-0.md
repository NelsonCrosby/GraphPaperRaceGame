MULTIPLAYER! This is what we've really wanted since the start.

Next is (debatably) the hard part - adding a GUI (with menus and stuff). This
 might take a while as we
 [decide what system to use](https://github.com/NelsonCrosby/GraphPaperRaceGame/issues/20).

[Pre-Release 2](https://github.com/NelsonCrosby/GraphPaperRaceGame/releases/tag/v0.2.0)
 release notes:

> Local multiplayer working nicely
>
> Issues resolved:
>
> - [#13]\: Make game use states
> - [#14]\: Local multiplayer
> - [#18]\: Better display of what options are available for the current turn
> - [#17]\: More clear indication of what player is currently active
> - [#16]\: Change number of players in local multiplayer
> - [#19]\: Get winning working on multiplayer
> - [#11]\: Natives not correctly resolving
>
> Known issues:
>
> - [#1]\: Having no start line causes crash
>   - Not a problem unless you start modifying the `.track` files provided in the JAR (which you are limited to for the time being).
> - [#9]\: (Linux-only) Switching windows crashes game
>   - An issue with LWJGL and X, not something we can fix unfortunately.
